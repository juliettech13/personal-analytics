LinkedIn — Direct API Token Required
LinkedIn Pages API requires an OAuth access token with r_organization_social permission. Paste your token into the LI_TOKEN constant at the top of the dashboard to go live.
To get a LinkedIn access token:
  1. Go to linkedin.com/developers → create an app (or use existing)
  2. Request r_organization_social + r_organization_admin permissions
  3. Generate a token via OAuth 2.0 (valid 60 days)
  4. Paste the token as LI_TOKEN in this file — LinkedIn data will load automatically
  • Followers & growth per page
  • Impressions & engagement rate
  • Likes, comments, shares
  • All 3 pages: Helicone · WWDW · Adult Camp

Twitter/X Organic — API Access Required
The X API v2 needs Basic tier access ($100/mo) to pull tweet analytics for @juliettech.
To unlock Twitter/X analytics:
  1. Go to developer.twitter.com and apply for Basic access ($100/mo)
  2. Generate a Bearer token and add it as X_TOKEN in this file
  3. Come back and refresh — this tab will populate automatically
  • Followers count & growth trend
  • Impressions & reach per tweet
  • Engagement rate (likes, retweets, replies)
  • Top performing tweets by engagement
